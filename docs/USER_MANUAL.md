# User Manual FreshMart Online

## Pembeli
1. Register sebagai pembeli.
2. Login.
3. Browse produk dan buka detail produk.
4. Klik tambah ke keranjang.
5. Masuk ke halaman keranjang, ubah jumlah barang jika perlu.
6. Checkout.
7. Upload bukti pembayaran.
8. Pantau status pesanan di halaman tracking.
9. Berikan review produk.

## Seller
1. Register seller.
2. Tunggu verifikasi admin.
3. Setelah diverifikasi, seller bisa tambah/edit/hapus produk.
4. Seller bisa mengelola stok produk.
5. Seller bisa melihat pesanan masuk dan mengubah status pesanan.

## Admin
1. Login admin.
2. Lihat dashboard statistik.
3. Verifikasi seller.
4. Kelola user, kategori, produk, pesanan, pembayaran, dan laporan.
5. Konfirmasi pembayaran agar status pesanan berubah otomatis.
